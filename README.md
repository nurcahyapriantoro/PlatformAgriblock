## 🌟 About

<table>
<tr>
<td>

**PlatformAgriblock** is a revolutionary blockchain-based solution designed to transform agricultural supply chains through enhanced transparency, traceability, and efficiency.

By leveraging the power of distributed ledger technology, we're building a secure ecosystem that connects all stakeholders in the agricultural value chain - from farmers and distributors to retailers and consumers.

<details>
<summary><b>🎯 Key Goals</b></summary>
<br>
  
- ✅ Create immutable and verifiable records for agricultural products
- ✅ Eliminate intermediaries and reduce transaction costs
- ✅ Empower farmers with fair pricing and direct market access
- ✅ Provide consumers with verified information about food origins and quality
- ✅ Build trust through transparent and traceable supply chains
</details>

</td>
</tr>
</table>

## 🚀 Getting Started

### Prerequisites

<details>
<summary>View Prerequisites</summary>
<br>

- Node.js (v18.0 or higher)
- npm (v9.0 or higher) 
- Next.js (v13.0 or higher)
- TypeScript (v5.0 or higher)
- Hyperledger Fabric (v2.5 or higher)

</details>

### Installation

<details>
<summary>View Installation Steps</summary>
<br>

1. **Clone the repository**
```bash
git clone https://github.com/nurcahyapriantoro/PlatformAgriblock.git
cd PlatformAgriblock
```

2. **Install dependencies**
```bash
cd backend > npm install
cd frontend > npm install
```

3. **Configure environment variables**
```bash
cp .env.example .env
```
Edit the `.env` file with your configuration details.

4. **Start the development server**
```bash
cd backend > npm run dev (Menjalankan Backend)
cd frontend > npm run dev (Menjalankan Frontend)
```

5. **Access the application at** `http://localhost:3000 or port 3001`

</details>
